# PokedexRemake
A remake of my [Pokedex](https://github.com/tinotusa/Pokedex) app. This is used for testing out the new PokeAPI wrapper package.

Pokeapi wrapper being used: [SwiftPokeAPI](https://github.com/tinotusa/SwiftPokeAPI).
